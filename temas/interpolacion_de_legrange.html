1. Interpolación de Lagrange

Conoce su origen Joseph-Louis Lagrange

2. Fundamento teórico ampliado

La interpolación de Lagrange construye un polinomio de grado ≤n que pasa por n+1 puntos (x_i, y_i) usando bases que son 1 en un punto y 0 en los demás.

P_n(x) = Σ_{i=0}^n y_i L_i(x), L_i(x) = Π_{j=0,j≠i}^n (x - x_j)/(x_i - x_j).

3. Término de error

El error se expresa como:

f(x) - P_n(x) = f^{(n+1)}(ξ)/( (n+1)! ) · Π_{i=0}^n (x - x_i), ξ en intervalo.

4. Pasos detallados

  1. Obtener puntos (x_i, y_i).
  2. Calcular polinomios base L_i(x) para cada i.
  3. Construir P_n(x) como combinación lineal Σ y_i L_i(x).
  4. Evaluar P_n(x) donde sea necesario.

5. Ejemplo resuelto

Interpolar f(x)=sin(x) en x={0,π/4,π/2}: y={0,0.7071,1}:

  • L_0(x) = (x - π/4)(x - π/2)/((0 - π/4)(0 - π/2)).
  • L_1(x) = (x - 0)(x - π/2)/((π/4 - 0)(π/4 - π/2)).
  • L_2(x) = (x - 0)(x - π/4)/((π/2 - 0)(π/2 - π/4)).
  • P_2(x)=0·L_0(x)+0.7071·L_1(x)+1·L_2(x).
  • En x=π/6≈0.5236, P_2(π/6)≈0.5 (sin(π/6)=0.5).

6. Código en Python y gráfica

import numpy as np
import matplotlib.pyplot as plt

def lagrange_basis(xi, x):
    basis = []
    n = len(xi)
    for i in range(n):
        def Li(x_val, i=i):
            prod = 1
            for j in range(n):
                if j != i:
                    prod *= (x_val - xi[j])/(xi[i] - xi[j])
            return prod
        basis.append(Li)
    return basis

# datos\xx = np.linspace(0, np.pi/2, 100)
xi = np.array([0, np.pi/4, np.pi/2])
yi = np.sin(xi)
Li = lagrange_basis(xi, xx)
Pn = sum(yi[i]*Li[i](xx) for i in range(len(xi)))

plt.figure()
plt.plot(xx, np.sin(xx), label='sin(x)')
plt.plot(xx, Pn, '--', label='Lagrange P_2')
plt.scatter(xi, yi, color='red')
plt.legend()
plt.title('Interpolación de Lagrange grado 2')
plt.show()

10. Conclusión y aplicaciones

La interpolación de Lagrange es directa y conceptualmente clara, pero requiere recomputar toda la interpolación al agregar puntos. Se usa en análisis simbólico y presentación de fórmulas explícitas.
