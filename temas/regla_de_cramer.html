1. Regla de Cramer

Conoce su origen Gabriel Cramer

2. Fundamento teórico ampliado

Para un sistema lineal A·x = b con A ∈ ℝn×n invertible, la Regla de Cramer expresa cada incógnita xi como el cociente de determinantes:
xi = det(Ai) / det(A), donde Ai es A con su i-ésima columna reemplazada por b.

xi = \frac{\det(A_i)}{\det(A)}, para i = 1,…,n.

3. Término de error y estabilidad

Computar determinantes para sistemas grandes es costoso y numéricamente inestable: los errores de redondeo pueden amplificarse en el cálculo de det(A) y det(Ai).

Coste O(n!) o O(n³) con LU; condición similar a inv(A), sensible a cond(A).

4. Pasos detallados del algoritmo

  1. Calcular det(A).
  2. Para cada i de 1 a n:
    1. Formar matriz Ai sustituyendo columna i por b.
    2. Calcular det(Ai).
    3. Obtener xi = det(Ai) / det(A).
  3. Armar vector x = (x1,…,xn)T.

5. Ejemplo resuelto

Resolver:
A = [[2, -1, 3], [1, 0, -2], [3, 1, 4]], b = [5, -1, 10].

  • det(A) = 2·det([[0,-2],[1,4]]) - (-1)·det([[1,-2],[3,4]]) + 3·det([[1,0],[3,1]]) = 2·(0·4 - (-2)·1) +1·(1·4 - (-2)·3) + 3·(1·1 - 0·3) = 2·2 +1·10 +3·1 = 4 +10 +3 = 17.
  • Formar A₁, A₂, A₃ y calcular determinantes:
    • det(A₁)=det([[5,-1,3],[-1,0,-2],[10,1,4]]) = ... = 23 → x₁=23/17≈1.3529
    • det(A₂)=det([[2,5,3],[1,-1,-2],[3,10,4]]) = ... = 1 → x₂=1/17≈0.0588
    • det(A₃)=det([[2,-1,5],[1,0,-1],[3,1,10]]) = ... = 68 → x₃=68/17=4

6. Código en Python y gráfica

import numpy as np
import time
import matplotlib.pyplot as plt

def cramer(A, b):
    n = A.shape[0]
    detA = np.linalg.det(A)
    x = np.zeros(n)
    for i in range(n):
        Ai = A.copy()
        Ai[:,i] = b
        x[i] = np.linalg.det(Ai) / detA
    return x

# medir tiempo para distintos n usando matrices aleatorias
sizes = [2,5,10,20]
times = []
for n in sizes:
    A = np.random.rand(n,n)
    b = np.random.rand(n)
    t0 = time.time()
    cramer(A, b)
    times.append(time.time()-t0)

plt.figure()
plt.plot(sizes, times)
plt.xlabel('Tamaño n')
plt.ylabel('Tiempo (s)')
plt.title('Costo computacional Regla de Cramer')
plt.show()

10. Conclusión y aplicaciones

La Regla de Cramer es sencilla conceptualmente pero no práctica para grandes sistemas debido a su coste y estabilidad. Es útil en sistemas pequeños o al obtener fórmulas simbólicas.

Se aplica en teoría de sistemas, análisis simbólico y demostraciones de unicidad de soluciones.
