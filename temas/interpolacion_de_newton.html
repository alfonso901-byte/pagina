1. Interpolación de Newton por diferencias

Conoce su origen Isaac Newton

2. Fundamento teórico ampliado

La interpolación de Newton utiliza diferencias divididas para construir pólinomio incremental:

P_n(x) = f[x_0] + Σ_{k=1}^n f[x_0,…,x_k] Π_{j=0}^{k-1}(x - x_j)

donde f[x_i,…,x_{i+k}] son diferencias divididas.

3. Término de error

El error en x es:

f(x) - P_n(x) = f^{(n+1)}(ξ)/( (n+1)! ) · Π_{i=0}^n (x - x_i), ξ en intervalo

4. Pasos detallados

  1. Tomar puntos (x_i, y_i).
  2. Calcular tabla de diferencias divididas f[x_i], f[x_i,x_{i+1}], …
  3. Formar P_n(x) con fórmula incremental.
  4. Evaluar P_n(x) en valores deseados.

5. Ejemplo resuelto

Interpolar f(x)=ln(x) en x=1,2,4: y={0,0.6931,1.3863}:

  • Tabla diferencias divididas:
    • f[1]=0, f[2]=0.6931, f[4]=1.3863
    • f[1,2]=(0.6931-0)/(2-1)=0.6931
    • f[2,4]=(1.3863-0.6931)/(4-2)=0.3466
    • f[1,2,4]=(0.3466-0.6931)/(4-1)=-0.1155
  • P_2(x)=0 +0.6931(x-1) -0.1155(x-1)(x-2).
  • En x=3: P_2(3)=0.6931·2 -0.1155·2·1≈1.171 f(3)=ln3≈1.0986 error≈0.0724

6. Código en Python y gráfica

import numpy as np
import matplotlib.pyplot as plt

def diff_div(x, y):
    n = len(y)
    table = np.zeros((n,n))
    table[:,0] = y
    for j in range(1,n):
        for i in range(n-j):
            table[i,j] = (table[i+1,j-1] - table[i,j-1])/(x[i+j]-x[i])
    return table[0]

# datos
x = np.array([1,2,4])
y = np.log(x)
coeff = diff_div(x,y)

# construir polinomio
xx = np.linspace(1,4,100)
Pn = np.zeros_like(xx)
for k in range(len(coeff)):
    term = coeff[k]
    for j in range(k): term *= (xx - x[j])
    Pn += term

plt.figure()
plt.plot(xx, np.log(xx), label='ln(x)')
plt.plot(xx, Pn, '--', label='Newton P_2')
plt.scatter(x,y, color='red')
plt.legend()
plt.title('Interpolación Newton por diferencias')
plt.show()

10. Conclusión y aplicaciones

La forma de Newton es eficiente y estable para añadir nuevos puntos, ya que reutiliza diferencias previas sin rehacer todo. Es ampliamente usada en interpolación adaptativa.
