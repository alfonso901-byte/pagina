1. Método de Newton-Raphson

Conoce su origen Isaac Newton & Joseph Raphson

2. Fundamento teórico ampliado

El método de Newton-Raphson aproxima raíces de una función diferenciable f(x) mediante iteraciones usando la tangente en el punto actual xn. La fórmula básica es:

xn+1 = xn - f(xn) / f'(xn)

Parte de la expansión en serie de Taylor de f alrededor de xn y desprecia términos de orden superior.

3. Término de error y convergencia

El método converge cuadráticamente si f'(x*)≠0 y la semilla inicial está suficientemente cerca de la raíz x*. El error satisface:

|xn+1 - x*| ≈ C · |xn - x*|2, con C = |f''(x*) / (2 f'(x*))|

4. Pasos detallados del algoritmo

  1. Elegir función f(x) y derivada f'(x).
  2. Seleccionar estimación inicial x0 cercana a la raíz.
  3. Calcular xn+1 = xn - f(xn)/f'(xn).
  4. Verificar criterio de parada: |xn+1 - xn|< tol o |f(xn+1)|< tol.
  5. Repetir hasta cumplir tolerancia o iteraciones máximas.

5. Ejemplo resuelto

Encontrar √2 resolviendo f(x)=x2-2 con x0=1.5 y tol=1e-3.

  • Iteración 1: x1=1.5 - (0.25)/3 = 1.4167
  • Iteración 2: x2=1.4167 - (0.0069)/2.8333 = 1.4142
  • Iteración 3: x3=1.4142 - (6·10-6)/2.8284 = 1.4142 (tol)
  • … raíz aproximada: x≈1.4142

6. Código en Python y gráfica

import math
import numpy as np
import matplotlib.pyplot as plt

def newton_raphson(f, df, x0, tol=1e-5, max_iter=50):
    xs = [x0]
    for i in range(max_iter):
        x1 = xs[-1] - f(xs[-1]) / df(xs[-1])
        xs.append(x1)
        if abs(x1 - xs[-2]) < tol:
            break
    return xs

# Ejemplo f(x)=x^2-2, df(x)=2x
f = lambda x: x**2 - 2
df = lambda x: 2*x
its = newton_raphson(f, df, 1.5, tol=1e-3)
errs = [abs(x - math.sqrt(2)) for x in its]

plt.figure()
plt.semilogy(range(len(errs)), errs)
plt.xlabel('Iteración')
plt.ylabel('Error absoluto (log)')
plt.title('Convergencia de Newton-Raphson')
plt.show()

10. Conclusión y aplicaciones

Newton-Raphson ofrece convergencia cuadrática rápida cuando las hipótesis se cumplen, pero requiere cómputo de la derivada y puede divergir si la semilla está lejos de la raíz o f'≈0.

Es ampliamente usado en ingeniería y ciencias para solucionar ecuaciones no lineales con alta precisión y eficiencia.
