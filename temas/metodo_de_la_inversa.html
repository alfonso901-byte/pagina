1. Método de la Inversa

Conoce su origen Desarrollo en Álgebra Lineal Moderna

2. Fundamento teórico ampliado

El método de la inversa consiste en hallar A-1 de una matriz no singular A y resolver A·x=b como x = A-1·b. Se basa en llevar [A|I] a [I|A-1] mediante Gauss-Jordan.

Definición: A-1A = AA-1 = I.

3. Término de error y estabilidad

Calcular la inversa aumenta la sensibilidad a errores numéricos (condicionamiento). El error relativo |ΔA|/|A| se amplifica por el número de condición cond(A) = ||A||·||A-1||.

Error estimado: ||Δx||/||x|| ≤ cond(A)·(||ΔA||/||A||).

4. Pasos detallados del algoritmo

  1. Formar matriz aumentada [A|I].
  2. Aplicar Gauss-Jordan hasta obtener [I|A-1].
  3. Multiplicar A-1·b para obtener x.
  4. Verificar solidez: cond(A) baja para estabilidad.

5. Ejemplo resuelto

Sea A = [[4,7],[2,6]], b=[-1,3]. Hallar A-1 y x.

  • Determinante det = 4·6 - 7·2 = 10.
  • Inv(A) = (1/10)[ [6,-7],[-2,4] ] = [[0.6,-0.7],[-0.2,0.4]].
  • x = A-1·b = [[0.6,-0.7],[-0.2,0.4]]·[-1,3] = [-2.7,1.4].

6. Código en Python y gráfica

import numpy as np
import time
import matplotlib.pyplot as plt

def inversa(A):
    return np.linalg.inv(A)

def solve_inversa(A, b):
    Ainv = inversa(A)
    return Ainv.dot(b)

# medir tiempo inversa vs Gauss-Jordan
sizes = [50,100,200,400]
t_inv, t_gj = [], []
for n in sizes:
    A = np.random.rand(n,n)
    b = np.random.rand(n)
    t0 = time.time(); np.linalg.inv(A); t_inv.append(time.time()-t0)
    t0 = time.time(); solve_inversa(A, b); t_gj.append(time.time()-t0)

plt.figure()
plt.plot(sizes, t_inv, label='np.inv')
plt.plot(sizes, t_gj, label='Inversa+Multiplicación')
plt.xlabel('Tamaño n')
plt.ylabel('Tiempo (s)')
plt.title('Costo computacional Método de la Inversa')
plt.legend()
plt.show()

10. Conclusión y aplicaciones

El método de la inversa proporciona solución directa por multiplicación pero es costoso y sensible a errores (condicionamiento). Es útil para múltiples sistemas con misma A.

Se emplea en control, optimización y simulaciones donde se reutiliza A-1 para distintas b.
