1. Método de Euler

2. Fundamento teórico

Euler aproxima la solución de una EDO y' = f(t,y), y(t_0)=y_0, usando la derivada en el punto actual:

y_{n+1} = y_n + h·f(t_n,y_n)

3. Término de error y estabilidad

Es método de orden 1: error local O(h^2), global O(h). Condición de estabilidad depende de paso y región de estabilidad.

4. Pasos detallados

  1. Dividir [t_0,T] en pasos de tamaño h.
  2. Para n=0…N-1 calcular y_{n+1}=y_n + h f(t_n,y_n).
  3. Actualizar t_{n+1}=t_n + h.
  4. Repetir hasta T.

5. Ejemplo resuelto

EDO: y' = -2y, y(0)=1, h=0.1 hasta t=0.5.

  • Tabla:
    y_1=1+0.1*(-2*1)=0.8, y_2=0.8+0.1*(-2*0.8)=0.64, …
  • Valor exacto y(0.5)=e^{-1}=0.3679, Euler da y_5≈0.3277 (error≈0.04).

6. Código en Python y gráfica

import numpy as np
import matplotlib.pyplot as plt

def euler(f, t0, y0, h, N):
    t = np.linspace(t0, t0+N*h, N+1)
    y = np.zeros(N+1)
    y[0] = y0
    for n in range(N):
        y[n+1] = y[n] + h * f(t[n], y[n])
    return t, y

f = lambda t,y: -2*y
t,y = euler(f,0,1,0.1,5)
plt.figure()
plt.plot(t, y, 'o-', label='Euler')
plt.plot(t, np.exp(-2*t), label='Exacta')
plt.legend()
plt.title('Método de Euler')
plt.show()

10. Conclusión y aplicaciones

El método de Euler es simple pero de bajo orden; a menudo se usa como base para métodos de orden superior o para visualización rápida de soluciones de EDO.
