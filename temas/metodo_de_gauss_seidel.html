1. Método de Gauss-Seidel

Conoce su origen Carle David Seidel

2. Fundamento teórico ampliado

Gauss-Seidel mejora al método de Jacobi usando los valores más recientes en cada iteración. Descompone A = D + L + U y aplica:

xi(k+1) = (bi - Σjaijxj(k+1) - Σj>iaijxj(k))/aii.

3. Término de error y convergencia

Converge si A es diagonalmente dominante o simétrica definida positiva. La velocidad suele ser mayor que Jacobi: convergencia lineal pero con constante menor.

4. Pasos detallados del algoritmo

  1. Separar A en D, L y U.
  2. Inicializar x(0).
  3. Para cada iteración k y para i=1…n:
    1. Calcular xi(k+1) usando fórmula de arriba.
  4. Verificar criterio: ||x(k+1) - x(k)||< tol.
  5. Repetir hasta convergencia.

5. Ejemplo resuelto

Resolver 4x + y + z = 7, x + 3y + z = -8, x + y + 5z = 6, x(0)=(0,0,0), tol=0.01.

  • Iteración 1: x1=(7-0-0)/4=1.75; y1=(-8-1.75-0)/3=-3.25; z1=(6-1.75-(-3.25))/5=1.3.
  • Iteración 2: x2=(7-(-3.25)-1.3)/4=2.1; y2=(-8-2.1-1.3)/3=-3.8; z2=(6-2.1-(-3.8))/5=1.34.
  • … hasta x≈[1, -2, 1].

6. Código en Python y gráfica

import numpy as np
import matplotlib.pyplot as plt

def gauss_seidel(A, b, x0, tol=1e-3, max_iter=100):
    n = A.shape[0]
    x = x0.copy()
    errs = []
    for k in range(max_iter):
        x_old = x.copy()
        for i in range(n):
            sigma = np.dot(A[i,:i], x[:i]) + np.dot(A[i,i+1:], x_old[i+1:])
            x[i] = (b[i] - sigma) / A[i,i]
        err = np.linalg.norm(x - x_old, ord=np.inf)
        errs.append(err)
        if err < tol:
            return x, errs
    return x, errs

# ejemplo
A = np.array([[4,1,1],[1,3,1],[1,1,5]])
b = np.array([7,-8,6])
x0 = np.zeros(3)
x, errs = gauss_seidel(A, b, x0)

plt.figure()
plt.plot(errs)
plt.yscale('log')
plt.xlabel('Iteración')
plt.ylabel('Error ∞ (log)')
plt.title('Convergencia Gauss-Seidel')
plt.show()

10. Conclusión y aplicaciones

Gauss-Seidel suele converger más rápido que Jacobi al usar valores nuevos inmediatamente. Es sencillo de implementar y efectivo para sistemas dispersos y en precondicionamiento.
