1. Método de Gauss-Jordan

Conoce su origen Carl Friedrich Gauss & Wilhelm Jordan

2. Fundamento teórico ampliado

El método de Gauss-Jordan extiende la eliminación gaussiana para llevar la matriz aumentada [A|b] a forma reducida por filas (RREF), donde A se convierte en la matriz identidad. De este modo, las soluciones aparecen directamente en el vector b.

Operación hasta RREF: Aumentar pivotes a 1 y eliminar en toda columna (superior e inferior).

3. Término de error y estabilidad

Al igual que Gauss, depende de pivoteo parcial para estabilidad. RREF puede intensificar errores de redondeo al realizar más operaciones de fila.

Complejidad O(n³) y factor de crecimiento controlado con pivoteo parcial.

4. Pasos detallados del algoritmo

  1. Formar matriz aumentada [A|b].
  2. Para k=1…n:
    1. Seleccionar pivote akk, pivoteo parcial si necesario.
    2. Dividir fila k por akk para hacer pivote=1.
    3. Para i≠k, restar aik·(fila k) de fila i para eliminar columna k.
  3. Al final, la matriz izquierda es identidad y b contiene solución única.

5. Ejemplo resuelto

Resolver:
A = [[2,1,-1],[ -3,-1,2],[ -2,1,2 ]], b=[8,-11,-3] usando Gauss-Jordan.

  • Paso 1: Augmentar y pivotear, dividir fila 1 por 2.
  • Paso 2: Eliminar columna 1 en filas 2 y 3.
  • Paso 3: Pivotear fila 2, dividir y eliminar en demás filas.
  • Paso 4: Repetir para fila 3 → solución x=[3,2,-1].

6. Código en Python y gráfica

import numpy as np
import time
import matplotlib.pyplot as plt

def gauss_jordan(A, b):
    M = np.hstack((A.astype(float), b.reshape(-1,1).astype(float)))
    n = len(b)
    for k in range(n):
        # pivoteo parcial
        i_max = np.argmax(abs(M[k:,k])) + k
        if i_max != k:
            M[[k,i_max]] = M[[i_max,k]]
        # pivote a 1
        M[k] = M[k] / M[k,k]
        # eliminar resto de columna
        for i in range(n):
            if i != k:
                M[i] = M[i] - M[i,k] * M[k]
    return M[:,:-1], M[:,-1]

# medir tiempo
sizes = [50,100,200,400]
times = []
for n in sizes:
    A = np.random.rand(n,n)
    b = np.random.rand(n)
    t0 = time.time()
    gauss_jordan(A,b)
    times.append(time.time()-t0)

plt.figure()
plt.plot(sizes, times)
plt.xlabel('Tamaño n')
plt.ylabel('Tiempo (s)')
plt.title('Costo computacional Gauss-Jordan')
plt.show()

10. Conclusión y aplicaciones

Gauss-Jordan simplifica la obtención de la solución directa sin sustitución regresiva. Aunque más costoso que Gauss, resulta útil para obtener inversas de matrices y rangos.

Se aplica en estadística, ingeniería y álgebra computacional donde se precisan soluciones explícitas o matrices inversas.
