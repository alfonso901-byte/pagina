1. Eliminación Gaussiana

Conoce su origen Carl Friedrich Gauss

2. Fundamento teórico ampliado

La eliminación gaussiana transforma un sistema lineal A·x = b en uno equivalente triangular superior por operaciones elementales de fila (suma, intercambio y multiplicación). Después se resuelve por sustitución regresiva.

Operaciones de fila: Fi→Fi - (aij/ajj)·Fj, para j.

3. Término de error y estabilidad

La estabilidad numérica depende del pivoteo. Sin pivoteo, errores por cancelación pueden crecer; con pivoteo parcial se controla el factor de crecimiento.

Factor de crecimiento ρ ≤ 2n-1 sin pivoteo; con pivoteo parcial, ρ suele ≪ 2n-1.

4. Pasos detallados del algoritmo

  1. Para k=1…n-1, seleccionar pivote akk (con pivoteo parcial intercambiar filas).
  2. Para i=k+1…n, calcular m = aik/akk y restar m·(fila k) de fila i.
  3. Repetir hasta triangular superior U.
  4. Resolver U·x = c por sustitución regresiva.

5. Ejemplo resuelto

Resolver:
A = [[2,1,-1],[ -3,-1,2],[ -2,1,2 ]], b=[8,-11,-3].

  • Paso 1: sin pivoteo, m21=-3/2, m31=-2/2 → filas 2 y 3 actualizadas.
  • Paso 2: k=2, pivote a22=0.5, m32=updated/0.5 → fila 3.
  • Resultado U y c: U·x=c → x3= -1, x2=2, x1=3.

6. Código en Python y gráfica

import time
import numpy as np
import matplotlib.pyplot as plt

def gauss(A, b):
    A = A.astype(float).copy(); b = b.astype(float).copy()
    n = len(b)
    for k in range(n-1):
        # pivoteo parcial
        i_max = np.argmax(abs(A[k:,k])) + k
        if i_max != k:
            A[[k,i_max]] = A[[i_max,k]]; b[[k,i_max]] = b[[i_max,k]]
        for i in range(k+1,n):
            m = A[i,k]/A[k,k]
            A[i,k:] -= m*A[k,k:]
            b[i] -= m*b[k]
    # sustitución regresiva
    x = np.zeros(n)
    for i in range(n-1,-1,-1):
        x[i] = (b[i] - A[i,i+1:]@x[i+1:]) / A[i,i]
    return x

# medir tiempo para diferentes tamaños
sizes = [50,100,200,400]
times = []
for n in sizes:
    A = np.random.rand(n,n)
    b = np.random.rand(n)
    t0 = time.time()
    gauss(A,b)
    times.append(time.time()-t0)

plt.figure()
plt.plot(sizes, times)
plt.xlabel('Tamaño n')
plt.ylabel('Tiempo (s)')
plt.title('Costo computacional Eliminación Gaussiana')
plt.show()

10. Conclusión y aplicaciones

La eliminación gaussiana es el método fundamental para resolver sistemas lineales directos. Con pivoteo parcial es estable y de complejidad O(n³). Se emplea en ingeniería, ciencia de datos y simulaciones numéricas.

Sus variantes (LU, Cholesky) aprovechan estructuras y ofrecen eficiencia adicional.
