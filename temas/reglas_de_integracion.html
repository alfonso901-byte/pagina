1. Reglas de Integración Numérica

Enfoque en Regla del Trapecio y Simpson (1/3 y 3/8)

2. Fundamento teórico ampliado

Las reglas de integración numérica aproximan ∫_a^b f(x) dx mediante sumas ponderadas de evaluaciones de f en subintervalos. Aprovechan polinomios de bajo grado en cada tramo.

Trapecio: ∫_a^b f ≈ (h/2)[f(a)+f(b)], Simpson 1/3: ∫_a^b f ≈ (h/3)[f(a)+4f(m)+f(b)], Simpson 3/8: ∫_a^b f ≈ (3h/8)[f(a)+3f(a+h)+3f(a+2h)+f(b)].

3. Término de error

- Trapecio: error ∝ -((b-a)^3/12n^2) f''(ξ).
- Simpson 1/3: error ∝ -((b-a)^5/2880n^4) f^{(4)}(ξ).
- Simpson 3/8: similar orden 4.

4. Pasos detallados del algoritmo

  1. Dividir [a,b] en n subintervalos de igual tamaño h=(b-a)/n.
  2. Para Regla del Trapecio compuesta:
    ∫ ≈ (h/2)[f(x_0)+2Σ_{i=1}^{n-1}f(x_i)+f(x_n)].
  3. Para Simpson 1/3 compuesta (n par):
    ∫ ≈ (h/3)[f(x_0)+4Σ_{odd}f(x_i)+2Σ_{even≠0,n}f(x_i)+f(x_n)].
  4. Para Simpson 3/8 (n múltiplo de 3):
    ∫ ≈ (3h/8)[f(x_0)+3Σ_{i mod 3≠0}f(x_i)+2Σ_{i mod 3=0, i≠0,n}f(x_i)+f(x_n)].

5. Ejemplo resuelto

Calcular ∫_0^1 e^{-x^2} dx usando n=4:

  • h=0.25, x_i={0,0.25,0.5,0.75,1}.
  • Trapecio: T ≈ 0.5*0.25*[f(0)+2Σ f(x_i)+f(1)] ≈ 0.7468.
  • Simpson 1/3 (n=4 par): S ≈ (0.25/3)[f(0)+4(f(0.25)+f(0.75))+2f(0.5)+f(1)] ≈ 0.7468.
  • Simpson 3/8 (n=3) no aplica directamente para n=4, usar n=3 o ajuste.

6. Código en Python y gráfica

import numpy as np
import matplotlib.pyplot as plt

def trapecio(f, a, b, n):
    h = (b-a)/n
    x = np.linspace(a,b,n+1)
    y = f(x)
    return h*(y[0]/2 + y[-1]/2 + y[1:-1].sum())

def simpson13(f,a,b,n):
    h=(b-a)/n
    x=np.linspace(a,b,n+1)
    y=f(x)
    return h/3*(y[0]+y[-1] + 4*y[1:-1:2].sum() + 2*y[2:-2:2].sum())

# ejemplo
f=lambda x: np.exp(-x**2)
a,b=0,1
ns=[4,8,16,32]
T=[trapecio(f,a,b,n) for n in ns]
S=[simpson13(f,a,b,n) for n in ns]

plt.figure()
plt.plot(ns,T,'-o',label='Trapecio')
plt.plot(ns,S,'-o',label='Simpson 1/3')
plt.xlabel('n')
plt.ylabel('Aproximación')
plt.legend()
plt.title('Convergencia Integración Numérica')
plt.show()

10. Conclusión y aplicaciones

Las reglas del trapecio y Simpson ofrecen balances entre simplicidad y precisión. Simpson 1/3 es exacto para polinomios hasta grado 3. Son ampliamente usadas en ingeniería y ciencias.
