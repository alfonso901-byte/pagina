1. Aproximación por Mínimos Cuadrados

Conoce su origen Carl Friedrich Gauss

2. Fundamento teórico ampliado

El método de mínimos cuadrados ajusta una función (p. ej. línea recta) a datos (xi,yi) minimizando la suma de cuadrados de residuos ri=yi-f(xi).

Min S = Σ_{i=1}^m [y_i - Σ_{j=0}^n c_j φ_j(x_i)]^2

Para ajuste lineal φ0=1, φ1=x creamos sistema normal A·c=b.

3. Término de error

El valor mínimo Smin refleja calidad de ajuste. Su raíz cuadrada normalizada RMS = sqrt(Smin/m).

4. Pasos detallados del algoritmo

  1. Elegir base φj(x) (poli, polinomial).
  2. Formar matriz de diseño Φ (m×(n+1)): Φ_{ij}=φ_j(x_i).
  3. Armar sistema normal: Φ^TΦ·c = Φ^T y.
  4. Resolver coeficientes c por Gauss o inversa.
  5. Calcular ajuste f(x)=Σ c_j φ_j(x) y residuos.

5. Ejemplo resuelto

Ajustar línea y=ax+b a datos: (0,1),(1,2),(2,2.9),(3,4.1).

  • Φ = [[1,0],[1,1],[1,2],[1,3]], y=[1,2,2.9,4.1].
  • Sistema normal:
    [4 Σx; Σx Σx^2]·[b;a] = [Σy; Σ(xy)].
  • Σx=6, Σx^2=14, Σy=10, Σxy=24.1 → [[4,6],[6,14]]·[b;a]=[10,24.1].
  • Solución: a≈1.03, b≈0.98 → f(x)=0.98+1.03x.

6. Código en Python y gráfica

import numpy as np
import matplotlib.pyplot as plt

# datos\xx = np.array([0,1,2,3])
y = np.array([1,2,2.9,4.1])
# diseño
A = np.vstack([np.ones_like(xx), xx]).T
# coeficientes mínimos cuadrados
c, *_ = np.linalg.lstsq(A, y, rcond=None)
# ajuste\xx_fit = np.linspace(0,3,100)
y_fit = c[0] + c[1]*xx_fit
# residuos
res = y - (c[0] + c[1]*xx)
rms = np.sqrt(np.mean(res**2))

plt.figure()
plt.scatter(xx, y, label='Datos')
plt.plot(xx_fit, y_fit, 'r-', label=f'y={c[1]:.2f}x+{c[0]:.2f}')
plt.title(f'Ajuste por mínimos cuadrados (RMS={rms:.3f})')
plt.legend()
plt.show()

10. Conclusión y aplicaciones

La aproximación por mínimos cuadrados es fundamental en análisis de datos y modelado estadístico. Permite estimaciones robustas cuando hay ruido.

Se aplica en regresión, calibración de instrumentos y machine learning.
