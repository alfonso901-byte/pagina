1. Método de Montante

Conoce su origen Enrique M. Montante

2. Fundamento teórico ampliado

El método de Montante, también conocido como método del determinante inverso, resuelve sistemas lineales sin usar fracciones intermedias. Opera sobre la matriz aumentada aplicando transformaciones en enteros, ajustando pivotes con un factor "montante" para mantener la integralidad.

Fórmula de actualización: a'ij = (akk·aij - aik·akj) / D, con D pivote anterior.

3. Término de error y estabilidad

Montante evita acumulación de errores de redondeo al diferir divisiones hasta el final. Aun así, para matrices mal condicionadas, el crecimiento de valores intermedios puede causar desbordes.

Crecimiento de coeficientes controlado por pivote inicial; sin divisiones frecuentes, reduce error de truncamiento.

4. Pasos detallados del algoritmo

  1. Formar matriz aumentada M = [A|b]. Inicializar D0 = 1.
  2. Para k=1…n:
    1. Seleccionar pivote P = M[k,k].
    2. Para i≠k, j≠k: M'[i,j] = (P·M[i,j] - M[i,k]·M[k,j]) / Dk-1.
    3. Actualizar filas i≠k y columnas j≠k con M'.
    4. Fijar M[i,k] y M[k,j] = 0, M[k,k] = P.
    5. Actualizar Dk = P.
  3. Al final, sistema diagonal y solución xi = M[i,n+1] / M[i,i].

5. Ejemplo resuelto

Resolver:
A = [[2,1,-1],[1,0,2],[0,3,-1]], b=[3,1,2] con método de Montante.

  • Inicial D0=1, M = [A|b]. Pivote k=1: P=2. Actualizar submatriz 2×2 y b.
  • Pivote k=2: P=M[2,2], actualizar con D1=2.
  • Pivote k=3 y calcular soluciones: x1=1, x2=0, x3=2.

6. Código en Python y gráfica

import numpy as np
import time
import matplotlib.pyplot as plt

def montante(A, b):
    n = A.shape[0]
    M = np.hstack((A.astype(int), b.reshape(-1,1).astype(int)))
    D_prev = 1
    for k in range(n):
        P = M[k,k]
        M_new = M.copy()
        for i in range(n):
            for j in range(n+1):
                if i!=k and j!=k:
                    M_new[i,j] = (P*M[i,j] - M[i,k]*M[k,j]) // D_prev
        for i in range(n): M_new[i,k] = 0
        for j in range(n+1): M_new[k,j] = M[k,j]
        M = M_new
        D_prev = P
    x = np.zeros(n)
    for i in range(n): x[i] = M[i,-1] / M[i,i]
    return x

# medir tiempo para tamaños pequeños (<50)
sizes = [10,20,30,40]
times = []
for n in sizes:
    A = np.random.randint(1,10,(n,n))
    b = np.random.randint(1,10,n)
    t0 = time.time(); montante(A,b); times.append(time.time()-t0)

plt.figure()
plt.plot(sizes, times)
plt.xlabel('Tamaño n')
plt.ylabel('Tiempo (s)')
plt.title('Costo Computacional Método de Montante')
plt.show()

10. Conclusión y aplicaciones

El método de Montante (Bareiss) ofrece una alternativa sin divisiones tempranas, reduciendo errores de truncamiento y trabajando en enteros hasta el final. Es útil en cómputo simbólico e implementación de librerías numéricas.

Se emplea en sistemas con coeficientes exactos y cuando el control de errores de flotante es crítico.
