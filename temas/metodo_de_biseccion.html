1. Método de Bisección

Conoce su origen Bernard Bolzano / Regula Falsi

2. Fundamento teórico ampliado

El método de bisección localiza raíces de una función continua f(x) en un intervalo [a,b] donde f(a) y f(b) tienen signos opuestos (teorema del valor intermedio). Divide iterativamente el intervalo por la mitad y selecciona el subintervalo que mantiene la condición de cambio de signo.

Condición: f(a)·f(b) < 0 garantiza existencia de al menos una raíz en (a,b).

3. Término de error y convergencia

Tras n iteraciones, el error en la aproximación xn satisface:

|xn - x*| ≤ (b-a)/2n

La convergencia es lineal y garantizada si f es continua en [a,b] y f(a)·f(b)<0.

4. Pasos detallados del algoritmo

  1. Elegir intervalo inicial [a,b] con f(a)·f(b)<0.
  2. Calcular punto medio c = (a+b)/2.
  3. Evaluar f(c). Si f(a)·f(c)<0 entonces b=c; de lo contrario, a=c.
  4. Comprobar criterios de parada: |b - a| < tol o |f(c)| < tol.
  5. Repetir hasta alcanzar tolerancia deseada.

5. Ejemplo resuelto

Encontrar raíz de f(x)=x3-x-2 en [1,2], tol=0.001.

  • Iteración 1: c=1.5, f(1.5)=-0.125 → f(1)·f(1.5)<0 → b=1.5
  • Iteración 2: c=1.25, f(1.25)=-0.7969 → b=1.25
  • Iteración 3: c=1.125, f(1.125)=-0.299 → b=1.125
  • Iteración 4: c=1.0625, f(1.0625)=0.099 → a=1.0625
  • Iteración 5: c=1.0938, f(1.0938)=-0.104 → b=1.0938
  • Iteración 6: c=1.0781, f(1.0781)=-0.0043 → b=1.0781
  • … hasta tol → x≈1.0790

6. Código en Python y gráfica

import math
import numpy as np
import matplotlib.pyplot as plt

def biseccion(f, a, b, tol=1e-5, max_iter=50):
    fa, fb = f(a), f(b)
    if fa * fb >= 0:
        raise ValueError("f(a) y f(b) deben tener signos opuestos")
    errores = []
    for i in range(max_iter):
        c = (a + b) / 2
        fc = f(c)
        errores.append(abs(b - a) / 2)
        if abs(fc) < tol or abs(b - a) / 2 < tol:
            return c, errores
        if fa * fc < 0:
            b = c
            fb = fc
        else:
            a = c
            fa = fc
    return c, errores

# Ejemplo f(x)=x^3-x-2
def f(x): return x**3 - x - 2
root, errs = biseccion(f, 1, 2, tol=1e-3, max_iter=20)
its = list(range(1, len(errs)+1))

plt.figure()
plt.plot(its, errs)
plt.xlabel('Iteración')
plt.ylabel('Error estimado')
plt.title('Convergencia del método de Bisección')
plt.show()

10. Conclusión y aplicaciones

El método de bisección es robusto y simple, garantizando convergencia global aunque sea linealmente lento. Es útil para hallar raíces cuando se conoce un intervalo con cambio de signo, ideal como paso inicial antes de métodos más rápidos (Newton, secante).
