1. Método Serie de Taylor

Conoce a su creador Brook Taylor

2. Fundamento teórico ampliado

La serie de Taylor aproxima una función f(x) alrededor de un punto a mediante un polinomio infinito construido con las derivadas de f en a. Para funciones suficientemente suaves, esta aproximación converge al valor exacto cuando el número de términos crece.

Expresión general:

f(x) = Σk=0∞ (f(k)(a)/k!)*(x-a)k

3. Término de error y convergencia

El residuo de Lagrange tras n términos está dado por:

Rn(x) = (f(n+1)(ξ)/(n+1)!)*(x-a)n+1, donde ξ ∈ (a,x).

La serie converge si Rn → 0 cuando n→∞. El radio de convergencia varía según la función (por ejemplo, ex: ∞; ln(1+x): |x|<1).

4. Pasos detallados del algoritmo

  1. Seleccionar f(x) y punto de expansión a.
  2. Calcular derivadas hasta orden n.
  3. Evaluar cada derivada en a.
  4. Montar polinomio de grado n.
  5. Estimar el residuo Rn para conocer precisión.

5. Ejemplo resuelto con sin(x)

Aproximar sin(x) en x=π/4 con 5 términos (a=0). Derivadas: sin, cos, -sin, -cos,...

  • sin(0)=0, cos(0)=1, -sin(0)=0, -cos(0)=-1, sin(0)=0

Serie:

sin(x)≈ x - x³/3! + x⁵/5! - x⁷/7! + x⁹/9!

En x=π/4≈0.7854 rad:

sin(π/4)≈0.7854 - 0.0807 + 0.0025 - 0.00005 + 0.000001≈0.7071

6. Código en Python y gráfica

import math
import numpy as np
import matplotlib.pyplot as plt

def taylor_series(x, n):
    return sum((x**k) / math.factorial(k) for k in range(n+1))

# Convergencia para e^x en x=1
x=1
ns = list(range(1,11))
errors = [abs(math.e - taylor_series(x,k)) for k in ns]

plt.figure()
plt.plot(ns, errors)
plt.xlabel('Número de términos')
plt.ylabel('Error absoluto')
plt.title('Convergencia de la serie de Taylor para e^x en x=1')
plt.show()

10. Conclusión y aplicaciones

La serie de Taylor es fundamental en ingeniería, física y computación para aproximaciones rápidas. Se usa en generación de funciones trigonométricas, exponenciales y soluciones de ecuaciones diferenciales en simulaciones.

Comprender el error y el radio de convergencia es clave para aplicar el método de forma segura y eficiente.
