1. Método de Jacobi

Conoce su origen Carl Gustav Jacob Jacobi

2. Fundamento teórico ampliado

El método iterativo de Jacobi resuelve A·x=b descomponiendo A = D + (L+U), donde D es la diagonal, L la parte estrictamente inferior y U la estrictamente superior. Cada iteración calcula:

x(k+1) = D-1 [b - (L+U) x(k)].

3. Término de error y convergencia

Converge si la matriz de iteración T = -D-1(L+U) satisface ||T||<1 en alguna norma. La convergencia es lineal, lenta comparada con Gauss-Seidel.

4. Pasos detallados del algoritmo

  1. Separar A en D, L y U.
  2. Escoger estimación inicial x(0).
  3. Para cada iteración k:
    1. Para cada i: xi(k+1) = (bi - Σj≠iaijxj(k))/aii.
  4. Verificar criterio: ||x(k+1) - x(k)||< tol o ||r||< tol.
  5. Repetir hasta convergencia.

5. Ejemplo resuelto

Resolver sistema: 4x + y + z = 7, x + 3y + z = -8, x + y + 5z = 6. x(0)=(0,0,0), tol=0.01.

  • Iteración 1: x1=(7 - 0 - 0)/4=1.75; y1=(-8-0-0)/3=-2.67; z1=(6-0-0)/5=1.2.
  • Iteración 2: x2=(7 - (-2.67) -1.2)/4=2.12; …
  • … hasta converger a x≈[1, -2, 1].

6. Código en Python y gráfica

import numpy as np
import matplotlib.pyplot as plt

def jacobi(A, b, x0, tol=1e-3, max_iter=100):
    D = np.diag(np.diag(A))
    R = A - D
    x = x0.copy()
    errs = []
    for k in range(max_iter):
        x_new = np.linalg.inv(D).dot(b - R.dot(x))
        errs.append(np.linalg.norm(x_new - x, ord=np.inf))
        if errs[-1] < tol:
            return x_new, errs
        x = x_new
    return x, errs

# ejemplo\ A = np.array([[4,1,1],[1,3,1],[1,1,5]])
 b = np.array([7,-8,6])
 x0 = np.zeros(3)
 x, errs = jacobi(A, b, x0)

plt.figure()
plt.plot(errs)
plt.yscale('log')
plt.xlabel('Iteración')
plt.ylabel('Error ∞ (log)')
plt.title('Convergencia Método Jacobi')
plt.show()

10. Conclusión y aplicaciones

Jacobi es fácil de implementar y paralelo, pero lento. Útil para sistemas dispersos y precondicionamiento en métodos iterativos avanzados.
