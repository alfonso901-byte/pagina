1. Interpolación con Splines

Conoce su origen Isaac J. Schoenberg

2. Fundamento teórico ampliado

Los splines cúbicos interpolan datos con polinomios de grado 3 en cada subintervalo [x_i, x_{i+1}], asegurando continuidad de valor, primera y segunda derivada en nodos.

Cada tramo S_i(x) = a_i + b_i (x-x_i)+ c_i (x-x_i)^2+ d_i (x-x_i)^3 con condiciones de suavidad.

3. Término de error

El error de spline cúbico está acotado por Máx |f^{(4)}(ξ)|·h^4/384, con h la distancia de nodos.

|f(x)-S(x)| ≤ (h^4/384)·max_{ξ} |f^{(4)}(ξ)|

4. Pasos detallados

  1. Ordenar nodos x_i y valores y_i.
  2. Armar sistema tridiagonal para segundas derivadas M_i:
    h_{i-1} M_{i-1}+2(h_{i-1}+h_i) M_i + h_i M_{i+1} = 6[(y_{i+1}-y_i)/h_i - (y_i - y_{i-1})/h_{i-1}].
  3. Resolver M por Thomas (tridiagonal).
  4. Determinar coeficientes a_i=y_i, b_i = (y_{i+1}-y_i)/h_i - (2M_i+M_{i+1})h_i/6, c_i=M_i/2, d_i=(M_{i+1}-M_i)/(6h_i).

5. Ejemplo resuelto

Interpolar puntos: (0,0),(1,1),(2,0) con spline natural (M_0=M_2=0):

  • h_0=1,h_1=1. Sistema: [2*2]M_1=6[(0-1)/1 - (1-0)/1]=-12 → M_1=-3.
  • Coeficientes tramo 0: a_0=0,b_0=1-3*1/6=0.5,c_0=0,d_0=(-3-0)/6=-0.5.
  • Tramo 1: a_1=1,b_1=(0-1)/1 - (2*(-3)+0)/6=-1+1=0,c_1=-3/2=-1.5,d_1=(0+3)/6=0.5.

6. Código en Python y gráfica

import numpy as np
import matplotlib.pyplot as plt
from scipy.interpolate import CubicSpline

x = np.array([0,1,2])
y = np.array([0,1,0])
cs = CubicSpline(x, y, bc_type='natural')
xx = np.linspace(0,2,100)
y_cs = cs(xx)

plt.figure()
plt.plot(xx, y_cs, label='Spline cúbico')
plt.scatter(x,y, color='red', label='Datos')
plt.legend()
plt.title('Interpolación con splines cúbicos')
plt.show()

10. Conclusión y aplicaciones

Los splines cúbicos ofrecen interpolación suave y estable, con baja oscilación, ideales para gráficos y modelado CAD. Se usan en ingeniería, animación y ajuste de curvas.
